# ferre-pooley

Sistema de gestión para ferretería: cotizaciones → notas de venta.

## Stack

- [Next.js](https://nextjs.org) (App Router) — desplegado en Vercel
- [Supabase](https://supabase.com) — Postgres, Auth y RLS
- [Resend](https://resend.com) — envío de correos (cotizaciones con PDF y link de aceptación)

## Desarrollo

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).
